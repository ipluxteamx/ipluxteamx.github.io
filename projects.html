aqua system

Aqua System is a hobby operating system i've been working on for a while.
i made it using the Cosmos OS framework/kit, which is a C# kit for making OSes.

this project is my second longest project to date (without pauses),
after Friday Night Ghostin' having debuted in the 16th of January 2023.

i'm still doing this project, but i'm not working on it as much as i used to,
mainly because of school and my mental health.

i am also out of ideas for this project, but i am trying to think of some.

the GitHub repo link is here,
go take a look at it if you want to, would make me happy.


friday night ghostin'

Friday Night Ghostin', or FNG' for short, is a mod for Friday Night Funkin'
i started doing in early-mid 2021, and cancelled it recently for
mental health and other reasons
(mainly the fact that no one was working on it for a long time).

it was done mainly on Psych Engine or forks of it,
and i was the only main and lead programmer for the mod.

the mod is about some ghosts with each ghost having a different personality.
there was also going to have a gloop demon called Gloopy that was going to be the
main vilain of the mod (v2).

there were other people working with me, like Samy, Luxane, and others.
